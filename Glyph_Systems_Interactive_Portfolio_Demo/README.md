# Glyph Systems Interactive Portfolio Demo

This is a self-contained static browser portfolio for **Glyph Systems — Logic Engine**.

## Open locally

Double-click `OPEN_PORTFOLIO.bat` on Windows. It starts a local web server and opens the portfolio in your default browser.

You may also open `index.html` directly, although the local launcher gives more reliable PDF and HTML embedding.

## GitHub Pages

Upload the complete folder without changing the relative paths. Configure GitHub Pages to publish from the repository root. `index.html` will become the portfolio home page.

## Folder structure

- `index.html` — interactive browser interface
- `documents/pdf/` — browser-ready versions of the seven core documents
- `documents/original/` — original editable Word documents
- `evidence/` — HTML evidence reports
- `assets/` — logo and visual assets
